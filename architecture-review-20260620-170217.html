Portfolio Architecture — Looping Scroll + Star-Map (2026-06-20)

Core Concept

ONE continuous downward-looping vertical scroll. Four full-viewport sections (Home/Work/About/Contact) + a Home clone. Constellation morphs continuously with scroll position. Seamless wrap via clone buffer + momentum-preserving scroll offset reset.

Three-Layer Stack

z-0: Persistent Background
Fixed aurora + constellation. Mounts once, never reloads. Morphs driven by scroll position.
  • app.component.html — <app-aurora> + <app-constellation>
  • aurora.component.ts — WebGL gradient (ogl)
  • constellation.component.ts — 8-star real constellations (Ursa Major/Orion/Leo/Cygnus), scroll-driven morph, rAF drift
z-1: Content Scroll
Four destination sections (Home/Work/About/Contact) + invisible Home clone at the bottom.
  • app.component.ts — scroll listener, cycle measurement, wrap reset (scrollTo(0, scrollY − wrapAt))
  • landingpage.component.ts — Home hero (CV button, socials)
  • work/about/contact.component.ts — destination content, scroll-reveal (fade + blur on enter)
  • scroll-stage.service.ts — carries real-time position (0..4) for constellation sampling
z-100: Navigation
Sticky top nav. Always transparent. Smooth-scrolls to any section.
  • site-nav.component.ts — nav links anchor to #dest-home/work/about/contact, smooth scroll on click

Key Files to Understand the Loop

  • docs/adr/0004-seamless-loop-clone-wrap.md — READ THIS FIRST. Full decision record.
  • CONTEXT.md — glossary ("Cycle", "Seam", "Loop clone", "One-direction")
  • app.component.ts — measure() sets wrapAt; onScroll() fires reset; positionFor() maps scroll→destination units
  • constellation.component.ts — renderAt() (line 191) — reads stage.position, interpolates stars, draws links with cross-fade
  • app.component.html — four <section #dest> + invisible clone with inert aria-hidden is-visible

Scroll Position Formula

position = (section_index + frac) where frac ∈ [0,1] across each section.
Wraps: position % 4 in the constellation; at scroll-top the morph is at 0 (Home), at bottom it's at 4 ≡ 0 (Home again).

Visible Tells (Known Residuals)

  • Native scrollbar thumb snaps top↔bottom each loop (the one unavoidable tell; hiding requires hiding native scrollbar)
  • Mid-morph cross-fade lines can read cluttered (Cygnus→Ursa Major at ~t=0.5) — tuning opportunity in renderAt dim/transit
